Fedora First Things to Do

Updated: 4-18-2020

Commands to Run After Install

1. Update Fedora

$ sudo dnf update && sudo dnf distro-sync -y && sync && sudo dnf autoremove

Reboot System -Autoremove Fix

  • Removes the initially-installed kernel in order to fix dnf autoremove
  • This will vary depending on the Fedora image version used
  • Find original Kernal Installed
  • sudo dnf list --installed 'kernel'*
  • sudo dnf remove *5.3.7* && sync

2. Change Hostname

$ hostnamectl status # view current hostname
$ hostnamectl set-hostname --static "deathstar" # set up new hostname

3. Install RPM fusion repository and get most recent update

$ sudo dnf update --refresh
$ sudo dnf install https://download1.rpmfusion.org/free/fedora/rpmfusion-free-release-$(rpm -E %fedora).noarch.rpm
$ sudo dnf install https://download1.rpmfusion.org/nonfree/fedora/rpmfusion-nonfree-release-$(rpm -E %fedora).noarch.rpm

4. Install Video Players and Codecs

$ sudo dnf install youtube-dl vlc -y
$ sudo dnf install \
    gstreamer-plugins-base \
    gstreamer1-plugins-base \
    gstreamer-plugins-bad \
    gstreamer-plugins-ugly \
    gstreamer1-plugins-ugly \
    gstreamer-plugins-good-extras \
    gstreamer1-plugins-good \
    gstreamer1-plugins-good-extras \
    gstreamer1-plugins-bad-freeworld \
    ffmpeg \
    mpv \
    gstreamer-ffmpeg

5. Themes

$ do dnf install numix-gtk-theme
$ do dnf install paper-icon-theme-1.5.0-4.fc31.noarch

$ sudo dnf install menulibre
  • Disable Wayland
cat '/etc/gdm/custom.conf' | grep '#WaylandEnable' > '/dev/null' && sudo sed -i 's/'#WaylandEnable'/'WaylandEnable'/g' '/etc/gdm/custom.conf' && cat '/etc/gdm/custom.conf' | grep 'WaylandEnable'

7. Misc Apps

$ sudo dnf install unzip p7zip p7zip-plugins wget
  • Fedy Install
# RPM Fusion
sudo dnf install https://download1.rpmfusion.org/free/fedora/rpmfusion-free-release-$(rpm -E %fedora).noarch.rpm https://download1.rpmfusion.org/nonfree/fedora/rpmfusion-nonfree-release-$(rpm -E %fedora).noarch.rpm

# Install fedy copr repository
sudo dnf copr enable kwizart/fedy

# Install fedy
sudo dnf install fedy -y
  • Nano Install and updates env variables
sudo dnf install 'nano' -y && export 'VISUAL=nano' && export 'EDITOR=nano' && echo -e 'export VISUAL=nano\nexport EDITOR=nano' | sudo tee '/etc/profile.d/editor-nano.sh' > '/dev/null' && cat '/etc/profile.d/editor-nano.sh' && mkdir -p ~/'.config/nano' && echo 'set nowrap' | tee ~/'.config/nano/nanorc' > '/dev/null' && cat ~/'.config/nano/nanorc'
  • Git Tools
sudo dnf install git-cola

8. Mozilla Firefox Settings

  • Enable hardware Acceleration
echo -e 'export MOZ_ACCELERATED=1\nexport MOZ_WEBRENDER=1\nexport MOZ_USE_XINPUT2=1' | sudo tee '/etc/profile.d/firefox-hwaccel.sh' > '/dev/null' && cat '/etc/profile.d/firefox-hwaccel.sh'
